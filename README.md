# Telegram-TreceBot
Based on https://github.com/yagop/node-telegram-bot-api and NodeJS

## Install
```
git clone <REPO>
cd <REPO>
npm install
```

# Config
```
cp data/config.sample.json data/config.json
cp data/frases.sample.json data/frases.json
```
Edit data/config.json to add TOKEN.

# Run
```
npm start
```
